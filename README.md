# 简介
本python脚本用于下载B站上的视频，默认清晰度是最高的<br>
B站上下载的文件是视频和音频分开的，所以要用ffmpeg进行合并

# 使用方法
1.安装python<br>
2.确保您已经安装好本脚本所需库/软件包<br>
  # 所需库/安装包
  · requests<br>
    用于网页访问<br>
  如果您没有安装，可以运行以下命令进行安装(前提是已经安装好Python)<br>
    · 安装requests<br>
      pip install requests<br>
      清华源:pip install -i https://pypi.tuna.tsinghua.edu.cn/simple requests<br>
3.填写你账号的cookie(不填也行)<br>
4.运行<br>

# 当然你也可以运行main.exe程序(推荐小白)
运行exe程序不需要安装python，也不需要安装任何库/软件包  <br>
文件可能会被杀毒软件误报，所以运行时请退出杀毒软件 (文件是没有病毒的) <br>
如果你不放心，可以麻烦一点，安装python和依赖包/库，然后运行代码文件
